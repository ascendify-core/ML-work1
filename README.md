
#  Bakery Demand Prediction System 

## 📌 Problem
Predict quantity of multiple bakery items based on number of customers.

## 🧠 Algorithm Used
Linear Regression (Multi-output)

## 📂 Dataset
- Customers
- Bread
- Cake
- Cookies

## ⚙️ Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
